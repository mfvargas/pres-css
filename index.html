CSS

    CSS (Cascading Style Sheets) es un lenguaje de diseño gráfico para describir la presentación de un documento escrito en un lenguaje de marcas (ej. HTML).

    Permite describir aspectos de diseño como colores, tipos y tamaños de letra, márgenes, alineamientos y muchos otros.

CSS

    Permite manejar la presentación separadamente del contenido, lo que brinda una mayor una flexibilidad en el diseño.

    Por ejemplo, varios archivos HTML pueden compartir una misma presentación al hacer referencia a un mismo archivo CSS.

    También posibilita un despliegue diferenciado de acuerdo a los tipos y tamaños de pantallas.

Historia de CSS

    Fue propuesto por Håkon Wium Lie, quien trabajaba en CERN con Tim Berners-Lee, en 1994.

    Al igual que HTML, CSS es un estándar de W3C. En la actualidad, la versión más utilizada es la 3 (CSS3).

¿En dónde especificar propiedades CSS?

Un navegador web busca propiedades CSS en:

  1. Elementos HTML (ej. body, h1, table, ...)
  2. La sección head de un documento HTML
  3. Archivos externos (.css)
  4. La configuración por defecto del navegador

Propiedades CSS en elementos HTML


                            <h1 style="color:blue;">Encabezado estilizado</h1>

¡Esta forma no es recomendada!

    No se cumple la separación entre contenido y estilo

Propiedades CSS en la sección head


                            <head>
                                <title>Ejemplo de CSS especificado en la sección head</title>
                                <style>
                                    h1 {
                                        color:blue;
                                    }
                                </style>
                            </head>

Propiedades CSS en archivos externos (.css)

    En la sección head del archivo HTML que va a usar los estilos, debe incluirse un elemento de tipo link:


                            <link rel="stylesheet" href="estilos.css">

Propiedades CSS en archivos externos (.css)

    Contenido de estilos.css:

Ejercicio

    Con base en el ejercicio de la lección anterior, construya un sitio web con información sobre los felinos de Costa Rica. El sitio debe tener al menos tres páginas:

    • index.html: con información general sobre los felinos de Costa Rica
    • manigordo.html: con información sobre el manigordo (Leopardus pardalis)
    • puma.html: con información sobre el puma (Puma concolor)

    Recuerde que puede obtener información de este sitio.

Ejercicio

    Cree un archivo llamado estilos.css que, para los tres archivos, especifique, por lo menos: el tipo y tamaño de letra, el color de fondo y el color de los encabezados.

    Agregue en cada página de especies, al menos un par de imágenes de cada una.

    En cada página, cree una barra de navegación que permita navegar a las otras dos páginas. Para esto utilice el elemento nav.

    Referencie las diferentes fuentes de información (textos, imágenes, etc.). Utilice los elementos cite y figure.

¿Cómo especificar colores en CSS?

  • Colores predefinidos (ej. red, green, ghostwhite, darkslategrey) (puede ver una lista en http://colours.neilorangepeel.com/).
  • Notación hexadecimal (ej. #ff00ff).
  • La función rgb() [ej. rgb(255, 0, 255)]
  • La función rgba() [ej. rgba(255, 0, 255, 0.5)]

Para sugerencias de colores en mapas, se recomienda revisar el sitio Color Brewer.

                                }
